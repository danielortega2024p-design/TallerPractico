Desarrollo Frontend

El Frontend es la parte de una aplicación web que los usuarios pueden ver e interactuar directamente.

Tecnologías principales

  1. HTML

    Define la estructura del contenido de una página web.

  2. CSS

    Permite controlar la apariencia visual del sitio.

  3. JavaScript

    Permite crear interfaces interactivas.
